# KIO v5

OpenStreetMap-overzicht voor Krimpen in & uit.

Nieuw in deze versie:
- vaste kleurgrenzen: groen 0–5, oranje 5–15, rood 15+ minuten;
- Algerabrug als één centrale status met drie satellieten voor fiets, auto en wisselstrook;
- vaartijden in het detailvenster van iedere veerverbinding;
- HLS-camera via een server-side proxy om CORS- en refererproblemen te omzeilen;
- geen geschatte wacht- of oversteektijden bij veren.

De actuele verkeersdata blijft afhankelijk van beschikbare NDW-koppelingen. Onzekere data wordt grijs weergegeven.
